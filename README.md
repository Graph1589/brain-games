## Description



Set of console mini-games.

## Installation

```
git clone https://github.com/Graph1589/brain-games
cd brain-games
make install
```
## List of Games



- Result of the expression:
```brain-calc```
- Number is even or not:
```brain-even```
- Find the GCD of numbers:
```brain-gcd```
- What number is missing:
```brain-progression```
- Number is prime or not:
```brain-prime```
